Title: Lanka Career Hub - Automated Job Aggregator & Social Media Bot 🤖

"මම මගේ පළවෙනි Python ප්‍රොජෙක්ට් එක විදිහට, රැකියා සොයන්නන්ට උදව් වෙන්න 'Lanka Career Hub' කියන Facebook පේජ් එක සම්පූර්ණයෙන්ම Automate කළා. 🦾

🚀 ප්‍රධාන විශේෂාංග (Key Features):

Multi-Source Scraping: ලංකාවේ ප්‍රධාන ජොබ් සයිට් 9ක් (TopJobs, Ikman, XpressJobs, etc.) සහ WhatsApp Channels නිරන්තරයෙන් පරීක්ෂා කිරීම.

GitHub Actions Integration: මගේ කොම්පියුටරය ඔන් එකේ තියන්න අවශ්‍ය නැතිව, GitHub Cloud එක හරහා සෑම පැයකටම වරක් ස්වයංක්‍රීයව ක්‍රියාත්මක වීම.

Meta Graph API: ජොබ් පෝස්ට් එකක් හමු වූ සැනින් පින්තූරයත් සමඟ ස්වයංක්‍රීයව Facebook පේජ් එකට පෝස්ට් කිරීම.

Smart Duplicate Detection: එකම ජොබ් පෝස්ට් එක දෙපාරක් වැටීම වැළැක්වීමට hash-based tracking පද්ධතියක් පාවිච්චි කිරීම.
